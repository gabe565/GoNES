## nesutil ines extract

Extract PRG/CHR ROM data from an INES ROM

```
nesutil ines extract ROM [flags]
```

### Options

```
  -c, --chr string   CHR ROM output file path
  -h, --help         help for extract
  -p, --prg string   PRG ROM output file path
```
