## nesutil chr encode

Encode a PNG file into NES CHR data

```
nesutil chr encode PNG [CHR data] [flags]
```

### Options

```
  -h, --help              help for encode
  -p, --palette strings   Palette to use. Must contain 4 hex colors. (default [000,555,AAA,FFF])
```
