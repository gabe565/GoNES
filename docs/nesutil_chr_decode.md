## nesutil chr decode

Decode NES CHR data into a PNG file

```
nesutil chr decode { ROM | CHR data } [output] [flags]
```

### Options

```
  -h, --help              help for decode
  -p, --palette strings   Palette to use. Must contain 4 hex colors. (default [000,555,AAA,FFF])
```
